# udemy-nodeJs-multiply-app

## Run

`npm i`

## Command line

Crear archivo

`node app crear -b 2`

Listar

`node app listar -b 2`

## Notes

Si al crear el archivo sale un error, ejecutar como administrador o crear la carpeta files en la raíz de la carpeta principal.
